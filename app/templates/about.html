About bgeo

This mapping application is intedended as a tool to add a bit of geographic understadning to the narratives of the Bible. This is a simple study aid and is not inspired. More details on the data sources is provided below, but nothing here is doctrine. Only the inspired word of God should be used to determine doctrine and dogma. If anything here contradicts with the Bible please let me know!

Data

The data used for locating places is sourced from www.openbible.info/geo. The sources given there are as follows:

  • ISBE
  • Eaton's
  • Smith's
  • Nave's
